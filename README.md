# Objection! Phoenix Wright Soundboard
## Introduction
This program allows you to get Phoenix Wright's iconic sounds from the _Ace Attorney_ series by pressing keyboard.
They are:
+ `A` key for "Objetion!"
+ `S` key for "Hold it!"
+ `D` key for "Take that!"

## Installation
Run
```
pip install -r requirements.txt
python Objection.pyw
```
